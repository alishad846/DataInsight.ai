# Dataset Guidelines

## Supported Formats
- CSV (.csv)
- Excel (.xlsx)

## Minimum Requirements
- Minimum 5 rows
- At least 2 columns
- Header row must exist

## Expected Column Types
- Numeric (int64, float64)
- Categorical (object)
- Date (datetime)
- Text columns

## Common Data Issues
- Missing values
- Duplicate rows
- Incorrect data types
- Columns with high null percentage
- Inconsistent date formats
- Extra whitespace in categorical columns
